# Modern Social Detroit — SEO Audit Dashboard

A clean, responsive HTML dashboard to visualize the SEO audit progress for Modern Social Dentistry, powered by live data from Monday.com.

## 🔗 Live Dashboard

[View Dashboard](https://kevinNunez-dev.github.io/Modern_Detroit_Social)

## 📊 What It Shows

- **Key Metrics** — Completed, In Progress, Not Started task counts + overall completion %
- **Status Doughnut Chart** — Visual breakdown of task statuses
- **Section Bar Chart** — Completion % per SEO section
- **Section Progress Table** — Per-group task breakdown
- **Full Task List** — All 40 tasks with status and assignee

## 🗂 SEO Sections Covered

1. Onboarding & Client Intake
2. Technical SEO
3. On-Page SEO
4. Local SEO & GMB
5. Content & Blogging
6. Analytics & Reporting

## 🛠 Tech Stack

- Pure HTML/CSS/JS — no build tools
- [Chart.js](https://www.chartjs.org/) for charts
- Data sourced from [Monday.com Board](https://rptclinic.monday.com/boards/18407794764)
